## 配置VSCode
1. 打开VSCode扩展商店，安装插件：
- Chinese (Simplified) (简体中文) Language Pack for Visual Studio Code - VSCode汉化
- C/C++ - C/C++ 插件
- clangd - 代码补全提示
- CMake Tools - CMake工具
2. `按Ctrl + ,`进入设置
- 找到扩展 -> C/C++ -> IntelliSense -> C_Cpp: AutoComplete，改为disable
